This plugin will enable sticky header with image, title, quantity option and add to cart button without touching any line of code. Only install and activate it, after activation it will automatically enable sticky header of WooCommerce on single product detail page. It's working only with WooCommerce plugin so main WooCommerce plugin should be activated.

== Description ==

Sticky WooCommerce Header Cart adds sticky header with basic WooCommercec cart options. A row to top header will be added with a thumbnail, title, quantity and add to cart button.

There is no need to add any code on your theme core file. Just install and activate it. 

### Features:
* No need to put any code on theme files
* No need  add_action or add_filter hooks 
* Sticky WooCommerce Header Cart on single product details page as per your requirements.

== Installation ==

How to install Sticky WooCommerce Header Cart.

*Using The WordPress Dashboard*

1. Navigate to the 'Add New' in the plugins dashboard
2. Search for 'Sticky WooCommerce Header Cart'
3. Click 'Install Now'
4. Activate the plugin on the Plugin dashboard
